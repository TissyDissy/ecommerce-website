# 電子商務網站

這是一個使用 React、TypeScript 和 Ant Design 構建的現代電子商務網站。

## 功能特點

- 產品瀏覽和搜索
- 購物車管理
- 用戶註冊和登錄
- 訂單管理
- 後台管理系統

## 技術棧

- React 18
- TypeScript
- Ant Design 5
- React Router 6
- Axios

## 開始使用

1. 克隆倉庫
```bash
git clone https://github.com/TissyDissy/ecommerce-website.git
```

2. 安裝依賴
```bash
npm install
```

3. 啟動開發服務器
```bash
npm start
```
